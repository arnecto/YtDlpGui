# YtDlpGui

Проста графічна оболонка на C++/Qt6 для [yt-dlp](https://github.com/yt-dlp/yt-dlp) — завантажуй відео та аудіо з YouTube (і не тільки) без командного рядка.

![Ліцензія](https://img.shields.io/badge/license-GPL--3.0-blue)

## Можливості

- Вибір якості: найкраще mp4, найкраще будь-яке, тільки аудіо (mp3), або власні аргументи yt-dlp
- Вибір відеокодека: без переваг (як є), H.264 з перевагою або примусовим перекодуванням (для сумісності з Adobe Premiere та іншими редакторами, які не тягнуть AV1), H.265, VP9
- Підтримка кук з браузера (Chrome, Firefox, Edge, Brave, Opera, Vivaldi) або з файлу `cookies.txt` — потрібно для вікових обмежень YouTube
- Поле для довільних додаткових аргументів yt-dlp
- Прогрес-бар і лог у реальному часі
- Завантаження субтитрів, підтримка плейлистів
- Налаштування зберігаються між запусками

## Завантажити

Готовий `.exe` для Windows шукай на сторінці [Releases](../../releases) — там завжди лежить остання зібрана версія, ставити Qt чи CMake не потрібно.

## Обов'язкові зовнішні інструменти

Ця програма — лише GUI-обгортка. Для роботи потрібні:

1. **[yt-dlp.exe](https://github.com/yt-dlp/yt-dlp/releases/latest)** — сам інструмент завантаження. Вкажи шлях до нього в полі "yt-dlp" при першому запуску.
2. **[ffmpeg](https://www.gyan.dev/ffmpeg/builds/)** — для об'єднання відео+аудіо та перекодування. Бажано мати в `PATH` або вказати папку в полі "ffmpeg".
3. **[Node.js](https://nodejs.org/)** (рекомендовано) — сучасний YouTube вимагає виконання JS-коду для обходу захисту (n-challenge). Без Node.js частина відео може не завантажуватись з попередженням `n challenge solving failed`.

## Збірка з коду

Вимоги: Qt 6.x, CMake 3.16+, компілятор з підтримкою C++17 (MinGW або MSVC на Windows).

```bash
git clone https://github.com/<твій-акаунт>/YtDlpGui.git
cd YtDlpGui
mkdir build && cd build
cmake -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH="C:/Qt/6.x.x/mingw_64" ..
cmake --build .
```

Або відкрий папку у VS Code з розширенням **CMake Tools** — вона підхопить конфігурацію автоматично.

## Вирішення проблем

**Відео не завантажується, помилка про вік/логін**
Вкажи файл cookies.txt (експортований розширенням типу "Get cookies.txt LOCALLY") або обери свій браузер у полі "Куки з браузера".

**`n challenge solving failed` / бракує форматів**
Онови yt-dlp до останньої версії та встанови Node.js. Один раз запусти з додатковим аргументом `--remote-components ejs:github`, щоб закешувати solver-скрипти.

**Файл не відкривається в Adobe Premiere**
YouTube часто віддає відео в кодеку AV1, який Premiere без додаткових плагінів не підтримує. Обери "H.264/AVC — примусово перекодувати" у списку "Відеокодек" (потрібен вказаний шлях до ffmpeg).
