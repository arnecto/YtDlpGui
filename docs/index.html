Що всередині

[gui]

Нативний Qt-інтерфейс

Жодного термінала — URL, папка збереження, формат і кодек в одному вікні. Прогрес-бар і лог у реальному часі.

[cookies]

Вікові обмеження YouTube

Підключення кук з браузера (Chrome, Firefox, Edge, Brave) або з файлу cookies.txt — для відео, що вимагають входу в акаунт.

[codec]

Контроль над кодеком

YouTube часто віддає AV1, який не тягнуть Adobe Premiere та інші редактори. Перевага H.264 або примусове перекодування — одним пунктом меню.

[ci]

Прозора збірка

Кожен реліз збирається автоматично з вихідного коду через GitHub Actions. Ніякого закритого бінарника «повір мені».
